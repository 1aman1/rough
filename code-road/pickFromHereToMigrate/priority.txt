────────────────────────────────────────────────────────────────────────────────
 PATTERNS — Learn these templates. They cut across all tiers.
────────────────────────────────────────────────────────────────────────────────

  Prefix Sum                two-pass trick; range sum in O(1) after O(n) build
  Fast & Slow Pointer       Floyd cycle detection; also finds middle of list
  BFS on Grid               4-dir / 8-dir; multi-source BFS start all sources at once
  DFS + memoization         top-down DP; cache on (index, state) tuple
  Binary search on answer   when asked "minimum X such that condition holds"
  Monotonic stack           next greater / smaller element in O(n)
  Backtracking template     choose → recurse → unchoose; prune early
  Top-K with min-heap       maintain heap of size K; pop when oversized
  Merge intervals           sort by start; extend or push based on overlap
  Traversal order choice    pre-order = build/copy; in-order = sorted BST; post-order = delete/height
